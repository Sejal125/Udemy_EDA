# Udemy Courses Exploratory Data Analysis
Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a real-world Udemy courses dataset using Python.
The objective of this analysis is to uncover trends, patterns, and insights related to course categories, subscribers, pricing, reviews, course levels, and publishing trends.

The project demonstrates practical data analysis skills including:

Data cleaning
Handling missing values
Removing duplicate records
Data visualization
Trend analysis
Aggregation and grouping operations

Key Insights
Paid courses generally contain more lectures compared to free courses.
Beginner-level courses are the most commonly offered courses on Udemy.
Python-related courses have a large subscriber base and high popularity.
Certain subjects dominate the platform in terms of course count and engagement.
The number of published courses increased significantly during specific years.
