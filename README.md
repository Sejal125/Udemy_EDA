# 📊 Udemy Courses Exploratory Data Analysis (EDA)

## 🚀 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on a real-world Udemy courses dataset using Python.

The goal of this analysis is to discover meaningful insights related to:
- 📚 Course categories
- 💰 Pricing trends
- 👨‍🎓 Subscribers
- ⭐ Reviews
- 🎯 Course levels
- 📅 Publishing trends

This project helped me strengthen my practical understanding of:
- Data Cleaning
- Data Analysis
- Data Visualization
- Trend Identification
- Insight Extraction
